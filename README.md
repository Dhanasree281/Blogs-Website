# Blogs Website (React.js)

A responsive Blogs Website built using React.js that allows users to view and read blog posts with a clean and user-friendly interface. The project focuses on frontend development, component-based architecture, and smooth navigation.

---

## Features
- Display list of blogs with title and summary  
- View full blog content on selection  
- Responsive and clean UI  
- Component-based design using React  
- Easy navigation between pages  

---

## Tech Stack
- **Frontend:** React.js, JavaScript  
- **UI:** HTML, CSS  
- **Tools:** Git, GitHub, VS Code  

---

## How It Works
1. Blog data is rendered using reusable React components.  
2. Users can browse blogs and open individual posts.  
3. React state and props manage data flow and UI updates.
